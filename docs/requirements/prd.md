## Критерии приемки MVP

Для успешного релиза MVP должны быть выполнены все MUST HAVE функции:

- [ ] Проект настроен и работает на iOS и Android
- [ ] UI Kit создан и используется во всех экранах
- [ ] Аутентификация работает (логин/пароль + биометрия)
- [ ] Витрина продуктов отображается корректно
- [ ] Портфель активов показывает актуальные данные
- [ ] Чат с менеджером работает в реальном времени
- [ ] Unit test coverage ≥ 70%
- [ ] Integration tests для ключевых сценариев
- [ ] Нет критических багов
- [ ] Performance requirements соблюдены

## Примечания

1. **Оценки являются приблизительными** и могут быть скорректированы после начала разработки
2. **Зависимости от бэкенда** могут повлиять на сроки реализации
3. **Открытые вопросы** в PRD должны быть решены до начала разработки соответствующих функций
4. **Story Points** используют шкалу Фибоначчи (1, 2, 3, 5, 8, 13, 21)
